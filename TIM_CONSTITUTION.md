# TIM CONSTITUTION (v1)

## 1. Core Principle
TIM is NOT an AI.
TIM is a minimal orchestrator.

All thinking, reasoning, and judgment MUST be done by the LLM.

---

## 2. Absolute Rules

1. TIM must NOT think
2. TIM must NOT analyze
3. TIM must NOT interpret deeply
4. TIM must NOT become complex

---

## 3. Context Rules

- Default = NO context
- Add context ONLY when clearly needed
- Maximum = 1 or 2 small contexts
- Avoid long prompts

If unsure → DO NOT add context

---

## 4. Design Philosophy

- Simplicity over intelligence
- LLM-first design
- Minimal intervention
- Avoid over-engineering

---

## 5. Forbidden

- AI-based classification
- Embedding-based decision
- Complex scoring logic
- Always-on memory injection

---

## 6. Definition

TIM = A system that maximizes LLM performance with minimal intervention
